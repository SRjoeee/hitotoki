## Recommended Compression Guidelines
- **Resolution**: 1920x1080 (16:9) is recommended.
- **Bitrate**: Keep it below 1.5 Mbps to ensure speedy loading on network entry.
- **Opacity/Grating**: Desaturate, and apply deep blacks contrast inside the raw videos for an optimal, eye-comfortable dark background.

# Hitotoki

ひととき (Hitotoki) は、日本各地に息づく生活、仕事、修行、技を、ひとときの滞在体験として届けるプライベートな滞在プログラムのプラットフォームです。

## 特徴 (Features)

- **シネマティックなUI/UX**: スクロールに連動する全画面の動画背景と、WebGLによるパーティクルエフェクト。
- **没入感のあるサウンド**: Web Audio APIを用いた環境音の自動生成（風の音、寺の鐘の音など）。
- **洗練されたタイポグラフィ**: 余白を活かした和風の美しい文字組みとレイアウト。
- **インタラクティブな滞在設計図**: 各滞在プログラム（猟師、寺修行、鍛冶など）のタイムラインや詳細をシームレスに表示。

## 技術スタック (Tech Stack)

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **3D / WebGL**: Three.js
- **Icons**: Lucide React

## ローカルでの実行方法 (Run Locally)

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. 開発サーバーの起動:
   ```bash
   npm run dev
   ```

3. ブラウザで確認:
   `http://localhost:3000` にアクセスしてください。
